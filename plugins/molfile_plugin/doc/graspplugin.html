This plugin reads and displays GRASP binary surface files This format uses triangles exclusively to to represent 3-D geometry. Short documents describing the GRASP surface file format can be found at these two sites:

  • http://www.msg.ucsf.edu/local/programs/grasp/html/Appendix%20A.html.
  • http://honiglab.cpmc.columbia.edu/grasp/grasp_contents.html#A.1

    Notes:

    Since GRASP surface files are a 'finished product' type of data, they can't be manipulated very much within VMD, though they can be displayed.

    Unused data At present, the plugin doesn't provide color data from the GRASP surface file, the surface is loaded using the active user-defined graphics drawing color.
