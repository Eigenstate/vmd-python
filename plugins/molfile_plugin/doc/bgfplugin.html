The bgf plugin reads files produced by Biograf

Notes:

Lone pairs and bonds fields:

The VMD BGF plugin currently ignores the lone pair and valence fields, since these have no equivalent piece of data in VMD.

Forcefield types

VMD equates the VMD "type" field with the BGF forcefield type; this means that DREIDING typing, for example, must be done manually (for example, with a tcl script).
