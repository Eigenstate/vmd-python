This plugin reads AutoDock Grid Map files, used by the package to store potential maps. More information on this file format can be found at http://autodock.scripps.edu/faqs-help/manual/autodock-3-user-s-guide/AutoDock3.0.5_UserGuide.pdf.

Notes:

Read-only
This plugin is used only to read files. VMD does not currently support writing this file type.

Unit Cell
This file format only supports orthogonal unit cells.
