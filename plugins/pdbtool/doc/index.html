The PDBTool plugin provides an easy-to-use interface for downloading structures, electron density maps, and viewing web entries provided by numerous databases on the internet, all of which can be indexed by the PDB accession code for the structure of interest.

The PDBTool currently provides links to the following web-based databases keyed by the PDB accession code:

  • Web PDB structure download to local file
  • Web PDB structure auto-load into VMD as a new molecule
  • Web query of entry in the SCOP database
  • Web query of entry in the CATH database
  • Web run of entry through the Procheck server
  • Web electron density map download from Uppsala EDM server
  • Web run of entry through the Uppsala Ramachandran server
  • Web query of entry in the PDB database
  • Web browse main PDB site

If you have suggestions for other databases which can be easily accessed by the PDB accession code, let us know.
