The TkCon plugin provides a Tk-based console which augments the standard VMD text console with numerous additional features. TkCon provides command history, path/proc/variable name expansion, multiple interpreter consoles, captures stdout and stderr, character and proc highlighting, history searching, copy/paste between consoles, communicate with other tk interpreters, supports dynamically loadable extensions, electric character and proc highlighting. The latest version of TkCon supports attaching to itcl/Tcl 8 namespaces.

Documentation and Tutorials

  • TkCon is discussed in the VMD tutorial
  • The TkCon project home page provides more information about the features and usage of TkCon: http://tkcon.sourceforge.net/
