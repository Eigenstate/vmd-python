LabelÂ¶

Control the visibility and style of labels applied to the molecule. Again, not super useful without EGL rendering as labels appear in the render window.

add Add a label of the given type.
delete Delete a label
get_values Get label values for every frame in the molecule.
listall Get labels in the given label category.
set_visible Sets a label to be visible
text_size Sets text size for all labels.
text_thickness Sets text thickness for all labels.
