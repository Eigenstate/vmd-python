\newcommand{\DOCDESC} {
  The \VMD\ \DOCTITLE\ describes to run and use the molecular
visualization program \VMD.  This guide documents the usage of both
the graphical user interface and the text console interface for
displaying and grapically manipulating molecules, and describes how to
customize the appearance and behavior of \VMD\ for each user.}
