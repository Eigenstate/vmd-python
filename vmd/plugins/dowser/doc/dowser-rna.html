In order to work with RNA, Dowser needs additional dictionary definitions. Following the same approach used by Cameron Mura for DNA, we developed corresponding RNA dictionary entries that can be found below:

  • myrna_dict_5ter.db
  • myrna_dict_5pho.db (phosphorylated 5'-terminus)
  • myrna_parms.db

Details about how the dictionary has been developed can be found on the above files.
