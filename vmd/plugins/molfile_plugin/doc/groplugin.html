This plugin reads and displays molecular structure files in Gromos87 format, as recognized by GROMACS. More information can be found at http://www.gromacs.org/documentation/reference_3.1/online/gro.html.

Notes:

Read-only
This plugin is used only to read files. VMD does not currently support writing this file type.

Unused data
GRO files may contain an optional "velocity" block. This data is not used by VMD.

Compressed files
Compressed files will need to be expanded before this plugin can read them.
