This plugin reads electron density maps stored in the BRIX format, used by O and other programs. More information on this file format can be found at http://zombie.imsb.au.dk/~mok/brix/brix-1.html#ss1.2.

Notes:

Read-only
This plugin is used only to read files. VMD does not currently support writing this file type.

Unused data
The header of the BRIX file contains the value of the RMS deviation of the electron density map. This value is not used by VMD presently.
