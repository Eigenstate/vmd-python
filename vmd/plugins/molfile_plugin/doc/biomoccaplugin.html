This plugin volumetric occupancy maps created by the 'biomocca' program developed by the Computational Electronics Group at UIUC: http://www.ceg.uiuc.edu/ The biomocca map format is quite simple and constists of ASCII floating point numbers:

  • Origin of the map
  • integer X/Y/Z counts
  • voxel size in Angstroms
  • voxels follow in Z fastest, Y next fastest, X slowest

Notes:

Read-only
This plugin is used only to read files. VMD does not currently support writing this file type.
