This plugin reads molecular coordinate data stored in Insight II/Discover Cartesian Coordinate format. More information about this file format can be found here: http://hincklab.uthscsa.edu/~ahinck/html/soft_packs/msi_docs/insight980/insight/a_FF-CLASSIC.fcs.html.

Notes:

Read-only
This plugin is used only to read files. VMD does not currently support writing this file type.

Multiple molecules
CAR files can contain multiple molecule entries; only the first of these is read by the plugin. Multiple molecule support may be available in the future.

Periodic Boundary Condition (PBC) information
This plugin currently ignores any two-dimensional PBC information that may be present. 3D PBC info, however, is loaded into VMD.

Helix records
These records are also ignored by VMD.
