This plugin reads and writes the PSF ASCII structure file format used by CHARMM, CNS, NAMD, and X-PLOR.

Notes:

CHARMM extensions
The current versions of the PSF plugin support the Charmm31 CMAP, CHEQ, and EXT extensions when reading, but only the CMAP extension is supported for writing.

Bonds
Bond information is read and written by the PSF plugin.
