These plugins read the portable binary trajectory file formats used by GROMACS. The xtc file format is explained in detail at http://www.gromacs.org/documentation/reference_3.1/online/xtc.html, and more information on the trr file format can be found at http://www.gromacs.org/documentation/reference_3.1/online/trr.html.

Notes:

Read-only
These plugins are used only to read files. VMD does not currently support writing these file types.

Unused data
Velocities, forces, and energies stored in the trajectory file are ignored by VMD.
