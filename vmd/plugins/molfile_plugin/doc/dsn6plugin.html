This plugin reads electron density maps stored in the DSN6 format. More information on this file format can be found at http://www.uoxray.uoregon.edu/tnt/manual/node104.html.

Notes:

Read-only
This plugin is used only to read files. VMD does not currently support writing this file type.

Unused data
The DSN6 file format does not contain any data that is not read and used by VMD.

Byte-order
DSN6 files are created using the native byte-order (e.g., "little-endian" or "big-endian") of the machine used to create them. This plugin can read DSN6 maps created with either byte-order.
